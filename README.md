# Anime → Cinema (working title)

Single-page web app: upload a clip (e.g., anime) + choose a target “look” (preset image + optional prompt) → receive a transformed, more 3D/realistic/cinematic version.

## Goal (MVP)
- Upload video
- Pick a look preset (thumbnail grid)
- Start a render job
- Wait / refresh status
- Download / view result

## Non-goals (for MVP)
- Multi-shot editing timeline
- Full creative suite (keyframes, camera control)

## Status
Planning / model selection.
